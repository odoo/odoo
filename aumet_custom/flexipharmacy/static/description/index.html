Odoo 14 POS Pharmacy (Community)

POS Pharmacy (Community)

NOTE
1. This module is tested and working in Odoo community edition but not tested in Enterprise version.
2. We are supporting Ubuntu OS and not Windows OS.
3. We will not support if any third party customization or changes. We support only core module.
4. Not compatible with mobile and tablet screen.
5. This module works online only.

Features

  • RFID POS Login
  • Product Screen
  • Default Customer
  • Change Product UOM
  • Current Screen Lock
  • Bag Charges
  • Display Warehouse Quantity
  • Product Internal Transfer
  • Customer Birthday Reminder
  • Customer Anniversary Reminder
  • Assign SalesMan with Order
  • Close Session
  • Customer Screen
  • Order/line Note
  • Doctor Screen
  • Material Monitor
  • POS Promotion
  • Create Purchase Order
  • Money In/Out
  • Loyalty Programme
  • Gift Card
  • Gift Voucher
  • Wallet
  • Generate Different Barcode and Unique Internal Ref.
  • Order/Product Return (With Serial/Lot)
  • Customer Purchase History
  • Vertical Categories
  • Serial/Lot Numbers
  • Active Ingredients
  • Cross Selling
  • Optional Products
  • Automatic Refill Medicine
  • Doctor Commission
  • Delivery Management
  • Works Offline (Sale Operation without any feature)

Reports

Prerequisite Packages

  • sudo pip3 install python-barcode
  • sudo pip3 install simplejson
