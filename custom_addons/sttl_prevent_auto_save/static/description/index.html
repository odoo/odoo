Configuration in Settings

For configuring the feature, the user needs to navigate to Settings > Prevent Auto Save.

Enabling the feature for all masters

To enable the feature for preventing auto save for all masters, the user should check on Prevent Auto Save Globally.
This will prevent auto saving the records in both form and list views for all masters available in the instance.

Enabling the feature for specific masters

To enable the feature for preventing auto save for some specific masters, the user should check Prevent Auto Save for specific Model and add the masters from clicking Add a model.
This will prevent auto saving in both form and list views only for the masters the user has selected.
