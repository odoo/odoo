Sales Order Integration in Delivery Orders for Odoo 16

This module extends the Delivery Orders functionality in Odoo by adding a custom button to link Sales Orders with Delivery Orders. It allows quick navigation from a Delivery Order to its related Sales Order, only when the Delivery Order was generated from a Sales Order.

Features

Allowing easy access to the originating Sales Order from the Delivery Order.
A navigation button in the Delivery Order form lets users access the linked Sales Order, visible only if generated from a Sales Order.
Computes and displays the count of linked Sales Orders directly in the Delivery Order form, showing this count on the button.

Installation Instructions

  1. Download the Module
    • Download the Action Button in DO to Navigate to Linked SO module from the Odoo App Store
    • Save the downloaded module file (usually a .zip file) to your Odoo addons directory.
  2. Install the Module
    • Log in to your Odoo system as an Administrator.
    • Go to Apps > Update Apps List to refresh the list of available modules.
    • In the Apps menu, search for "Action Button in DO to Navigate to Linked SO".
    • Click the Install button next to the module.
  3. Troubleshooting
    • If you don't see the module in the Apps list: Ensure that you have activated developer mode by going to Settings -> Activate Developer Mode.
    • If the button does not appear in the Delivery Order: Verify that the Delivery Order is linked to a confirmed Sales Order. The button will only be visible when there is a linked Sales Order.
