FAQ

Is it possible to set limits on salary advances?

Yes, you can configure a percentage limit for salary advances in the salary structure.

Can I set a specific number of advance days?

Yes, you can define the minimum number of days before an employee can request a salary advance.

Can the module handle multiple salary advance requests?

Yes, the system can manage and track multiple salary advance requests for different employees.

How is the advance salary deducted from future payrolls?

The system automatically deducts the advance salary from the employee's future payslips based on the configured rules.
