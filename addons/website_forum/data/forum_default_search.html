How do I search?

Enter search terms in the search box that appears in the upper right corner of every page, and press Enter.

If you need to refine and narrow your search, here are some basic tips:

  • To search within specific tag(s), enter them in square brackets: [python] html searches for mentions of "html" within the [python] tag.
  • To find a specific phrase, or to search a string of special characters, enter it in quotes: like "ERP and eCommerce".
  • To search User posts: user:<user_id> training searches all of user posts for the word "training".
  • To exclude results from a tag, term or phrase, prefix it with - in your query: [python] -[css] includes only posts having python tag and exclude posts which having css tag.

Advanced Search Options:

We offer a number of advanced search operators to help you find very specific information.

Range Operators

To search for only questions that fall within a particular range for score, number of answers, or number of views, you can enter an upper or lower parameter, or a range.

  • score:-1 will both return posts with a score greater than or equal to -1
  • views:500 will return posts with a views greater than or equal to 500
  • answers:3 will return questions with a answer greater than or equal to 3

User Operators

You can also limit searches to a specific user's content (your own or someone else's). You will need the user id to search user's posts.

  • user:<user_id>returns only the posts from whichever user whose id you entered

OR operator

To combine results from multiple tags, separate the tag names (enclosed in square brackets) with the word "or": [python] or [html] returns questions tagged with either tag.
