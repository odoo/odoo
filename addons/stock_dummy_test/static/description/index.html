Stock Easy Internal Transfers

In some case is needed start a production with partial reservation because you can deliver your order in different parts. This module allow process a manufacturing order with partial reservation, producing multiple times until the order is completed. The module add a new compute field to show the quantity available to produce considering the number of the reservation in the products to consume. This field is considered in all process and you cannot produce more than this field indicates. This module don't allow the creation of negative quant, this mean you cannot force the production.
