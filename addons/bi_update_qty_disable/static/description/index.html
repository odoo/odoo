Hide Update Product Quantity Feature Odoo App

Update Product Quantity Feature Disable Odoo Apps

"Disable Update Quantity Feature Odoo App" helps users to prevent unauthorized modifications to the quantity of product. This app provides a simple solution for disabling the update quantity feature for certain users in Odoo, Currently in Odoo multiple users have access to update the quantity of products, this app provides an effective way to prevent every user from updating the product quantity, Allowed users can only update quantity for products.

Features

Disable Update Quantity Feature

Disable the update quantity feature for certain users.

Update Quantity User Access

Allowed users can only update product quantity.
Demo Request Video Tutorial

Disable Update Quantity Button

After installing module user can see "Update Quantity" button is invisible.

Users also can not create quantity for product by clicking "On Hand" button.

Update Quantity User Access

Users with "Onhand Quantity" access rights can only update quantity for product.

Update Quantity Button

Users with "Onhand Quantity" access rights can see "UPDATE QUANTITY" button and update quantity.

Create Quantity Button

User can see "CREATE" button and create quantity for product.
