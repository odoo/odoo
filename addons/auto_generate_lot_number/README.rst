Auto Generate Serial/Lot number
===============================
This app allows you to automatically Generate and manage serial/lot number for products while
confirming the purchase order.

Configuration
=============
- Additional configuration not required

Installation
============
- www.odoo.com/documentation/16.0/setup/install.html
- Install our custom addon
