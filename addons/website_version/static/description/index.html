Website Versioning

Save all the versions of your website with Website Versioning

Never loss any data on your website when you are editing pages. With Website Versioning you can save all the modification you realised and retrieve all of them through the versions.

Features

Quickly create different versions of your website, organize all versions with date or name and update all your website in just one click.

Save a modification as a draft version

People used to modify their website instinctively. Sometimes it is great but sometimes it is also bad. With Website Versioning, delete the bad and save the great through versions.

Publish a draft version you like

When you decide that a version seems to be the good one to illustrate your website, you can simply publish this version in one click.

Travel in the past

With Website Versioning, you can easily retrieve your old versions thanks to their name. Just choose the right one to begin the edition of a new great version.
