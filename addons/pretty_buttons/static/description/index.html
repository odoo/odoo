Pretty Buttons

Pretty Buttons is a customizable module for Odoo 16.0 that allows users to apply a stylish design to the buttons in their interface. This module provides the ability to enhance the visual appearance and user experience in Odoo applications.

Key Features

  • Beautiful button designs for your Odoo interface
  • Easy to configure and apply styles
  • Compatible with Odoo 16.0, 17.0
