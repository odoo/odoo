Presentation, Video, Documents

Publish your presentation, video, documents, infographics etc.

Odoo presentations lets you upload files privately or publicly in the following file formats: Power Point, PDF, Keynote, Video or Open Document presentations.

This module originally meant to be used for businesses to share slides among employees more easily. It also provides users the ability to rate, comment on, and share the uploaded content.
