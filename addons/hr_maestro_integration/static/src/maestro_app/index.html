📋 Quando ocorrer
🚘 Ticket perdido
1. Solicitar RG e CPF · 2. Verificar placa no sistema (F9) · 3. Preencher formulário de ticket perdido · 4. Cobrar tarifa máxima · 5. Registrar como "serviço"
🚧 Abrir cancela manualmente
Cloud Park: F7 no registro de saída → selecionar cancela → Abrir Cancela. Registrar motivo — é auditado pelo financeiro.
📋 QR Code já utilizado
Refazer a operação. Ao pedir para aproximar o QR Code, pressionar ENT e informar o valor da NF manualmente.
