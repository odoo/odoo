Restriction on Product Creation

This application is help to put restriction on product creation.



  Looking for Demo or Support?

1) Users View

Here user with rights of restriction on create product can see the boolean for Restriction on Create Product.

Navigation: Settings -> Users & Companies -> Users -> select your user -> Restriction on Create Product

2) pop-up Warning

Here if any user is restricted from creating the products then if user try to create any product then this app will show you warning pop-up.
