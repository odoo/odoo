Convert Purchase Order from Sale Order Odoo Apps

Odoo Sales Order Conversion to Purchase Orders App

Convert Purchase Order from Sale Order Odoo App designed to streamline the procurement process by automating the conversion of sales orders into purchase orders. This app enables the quick and accurate creation of purchase orders/RFQ for supplier/vendor based on the details of corresponding sales orders including Products, prices, quantity etc.

Convert Purchase from Sales

Create multiple purchase orders with one click, based on active sale orders.

Vendor Pricelist for Purchase

Purchase order uses vendor pricelist; if unavailable, defaults to product cost price.

Quick PO from Sales

Order creation loads product, price, quantity, and UOM from sale order.

Assign Vendor on Creation

Users can choose supplier and order date when creating a purchase order.

Create sale order

When you create sale order the "Create Purchase Order" button will appear in sale order.

Create Purchase Order Wizard

Create purchase order wizard with sale order line data, you can select vendor here.

Purchase Order Smart Button

User can esily access created purchase orders from sale order form view by clicking "Purchase Order" smart button, Also created purchase order linked in chatter.

View Purchase Order

User can View newly created purchase order by clicking on smart button.

Purchase Order with Vendor's Pricelist

Users with "Multi Branches" access rights can see and change multiple branch from header.
