# CAMTEL Core Module

**Minimal starter template for CAMTEL customizations**

## What's Included

- Basic module structure
- Empty models/ directory (ready for your models)
- Empty views/ directory (ready for your views)
- Security placeholder
- Main CAMTEL menu (ready to add submenus)

## Installation

```bash
./odoo-bin -c odoo.conf -d camtel_warehouse -i camtel_core
```

## Ready to Extend

Tell Claude exactly what functionality you want to add, one step at a time!

Examples:
- "Add a model for X"
- "Create a view for Y"
- "Add SMS functionality"
