Overview

A2A Digital 's Sport Management System envisions in giving a complete new experience to client in booking the sport services without complicated sign up and sign in process. The Sport Management System helps the customer in processing their online booking or booking directly to administrator for sport services. This module comes integrated with Odoo Accounting and Website, offering fully completed functionality sarting from the booking process till the completed payment.


NOTE : This Sport Management is available for public user to book which mean all user who access to the website are eligible to create the booking without sign up or sign in , Therefore the default timezone is Asia/Phnom_Penh ( In Cambodia Country ) and Cambodia shares the same time zone with
  • Vietnam
  • Thailand
  • Laos
  • Western Indonesia
  • Christmas Island
If you need this app to use outside of the above listing country, You will have a special personal support from developer for timzone configuration.

Chair

Go to Sport --> Court

Here you can create and edit details of each court.

Services

Go to Sport --> Services

List of Court services. You can create and edit more here.

Duration

Go to Sport --> Duration

List of Sport duration provided. You can create and edit duration of the sport here.

Orders

Go to Sport --> Sport Orders

Here you have all the orders of sport.

Booking

Go to Salon --> Bookings

Here you can see all the online bookings. You can either approve a booking or reject a booking.

Settings

Go to Salon --> Configuration --> Settings

Here you can assign the court, duration, activate the Banking QR Code Payment for online booking.

Configuration SMTP Email

Go to Salon --> Configuration --> SMTP Email Configuration

Configuration QRCode Payment

Go to Salon --> Configuration --> QR Code Payment

Here you can add the QR Code image, Banking Account ID & Account Name
