Product Image On Picking and Delivery/Receipt Report Odoo Apps

Picking Order Product Image Odoo App supports for both community as well as enterprise version. This odoo app helps to print product image in delivery order/receipt report of odoo. With this module user can see images of product in the picking line. it is easier to identify product via image in Odoo ERP Module.

Features

Product Image on Picking Order Line.

User can see product image on picking line of incoming shipment and delivery order.

Product Report on Piking Report.

User can see and print product image on picking line of incoming shipment and delivery order.
