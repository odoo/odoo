Build an Odoo 18 custom module named "smart_ecommerce_extension".

Features:

1. Extend product.template:
   - Fields: brand (char), model (char), specifications (html), video_url (char)
   - Methods: compute availability_status ("in_stock", "low_stock", "out_of_stock")

2. Extend website_sale:
   - Add product filters: brand, price_range, availability
   - Improve product grid card: second image on hover, stock badge
   - Add dynamic "estimated delivery date"

3. Add delivery rules:
   - Model: delivery.zone (name, cities, base_price, extra_price_per_kg)
   - Compute price based on product weight + city

4. API Endpoints (REST):
   - /smart/api/products
   - /smart/api/cart
   - /smart/api/checkout
   - /smart/api/payment/callback
   Response format: JSON

5. Create marketplace_seller model:
   - Fields: user_id, company_name, kyc_doc, state (draft/approved/suspended)
   - Portal views: dashboard, orders, payments

Deliver folder structure, __manifest__.py, models, controllers, views, security, data.
