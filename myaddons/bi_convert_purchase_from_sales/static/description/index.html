Convert Purchase Order from Sale Order Odoo Apps

This module gives feature to create Purchase Order / RFQ request to your supplier/vendor for product you want from them based on sales request(only if sale order not in Done or Cancel state).

Features

Convert Purchase from Sales.

You can create multiple purchase order to the vendor easily with one click, based on sale order only if not in done or cancel state.

Vendor pricelist applied on purchase.

Create purchase order it will take vendor's pricelist on purchase order and if pricelist is not define on vendor it will take cost price of product.
Demo Request Video Tutorial

Create sale order

When you create sale order the "Create Purchase Order" button will appear in sale order.

Create Purchase Order Wizard

Create purchase order wizard with sale order line data, you can select vendor here.

Purchase Order with Vendor's Pricelist

You can see that new purchase order is created based on product price of selected vendor's pricelist.

Purchase Order with Product Cost Price

When pricelist for selected vendor is not available then purchase order is created based on product cost price.
