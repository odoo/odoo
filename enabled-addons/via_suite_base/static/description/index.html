Requirements

  • Odoo 19.0 Community Edition
  • Python 3.12+
  • PostgreSQL 13+
  • Keycloak instance
  • AWS account (SES + S3)

Configuration

All configuration is done via environment variables:

  • VIA_SUITE_KEYCLOAK_* - Keycloak OAuth settings
  • VIA_SUITE_SMTP_* - Amazon SES configuration
  • VIA_SUITE_S3_* - S3 storage settings
  • VIA_SUITE_SENTRY_DSN - Sentry error tracking
