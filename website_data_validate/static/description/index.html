Validate Billing & Shipping Information

Validate Billing & Shipping Information entered by user on ecommerce website before submitting

The standard e-commerce module validates billing and shipping information only after submitting. This is ok but to save time and also guide the user on what needs to entered in every mandatory field, this module will be useful in doing that.

Its based on Jquery validation plug-in which makes it easier and flexible to customize the validation methods and messages that appear on the screen when invalid entries are keyed in by the user.

The standard e-commerce module does not give give back error messages when an input is invalid but. This module is addressing that issue by displaying error messages to the user hence making is more user friendly.

additionally, special input fields like phone, email, transaction codes, Credit card numbers, Bank Identification codes, Countries and states can be fully validated using this module.

Currently its validating the following billing and shipping information but can be extended easily to validate other type of information:

  • Name
  • Email
  • Street address
  • Phone Number
  • City
  • Country
