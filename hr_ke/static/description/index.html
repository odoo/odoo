Payroll Processing

The Payroll processing is done efficiently, accurately and in full compliance with Kenyan tax laws

The Kenya specific payroll features include but not limited to the following:

  • User defined Formula for determining the P.A.Y.E Tax based on employee earnings and benefits
  • User defined Formula for determining the Withholding Tax based on employee earnings
  • User defined formula for determining the taxable value of Benefits (Non-cash) per employee. Such Benefits include:
    • Housing
    • Servants
    • Company vehicle
    • Free Electricity
    • Free water
    • Free Telephone
  • User defined Formula for determining the "Allowed Deductions" (Pre-Tax Deductions) per employee; such as:
    • NSSF deductions based on latest NSSF rates of 2015
    • Deduction of Contributions to Home Ownership Savings Plan (HOSP)
    • Deduction of Contributions to defined Pension Fund/Provident Fund Scheme
    • Deductions of Interest paid for mortgage or Loans under the provisions of "Owner Occupied Interest" - OCI
  • User defined Formula for determining Tax Relief per employee; such relief include:
    • Insurance Relief
    • Personal Relief
  • User defined Formula for determining Post-Tax Deductions per Employee; such as:
    • NHIF contributions
    • Higher Education Loan Board (HELB) loan payments
    • Company Loan Payments
    • SACCO Loans
    • Contributions to SACCO/Chamas etc
    • Salary Advances
    • Any other deductions
  • You can easily define different kinds of salary computation methods to be used to run payroll for any kind of employee contracts such as:
    • Normal Employee Contract (paid monthly rate )
    • Casual Worker Contract (paid on daily rate)
    • Casual Worker Contract (paid on hourly rate)
    • Casual Worker Contract (paid on weekly rate)
  • Note that for each of the above contract types, you will flexibly pick the applicable taxation method to use
  • You can generate monthly, weekly or daily payslips from approved employee timesheets. The employee will initiate a timesheet and send it manager for approval. Once approved, the payslip is automatically generated and awaits confirmation from HR officer before payment is made
  • The payroll is linked to the accounting module so that when a payslip is confirmed by HR officer, the relevant journal entries are instantly generated, which can then be posted to reflect in relevant accounting books such as profit and loss account and the balance sheet
  • You have access to multiple kinds of instant reports including:
    • Monthly, weekly, daily payslips report
    • Muster Payroll report
    • NSSF Contrubutions report
    • NHIF Contrubutions report
    • P.A.Y.E Contrubutions report
    • KRA specific reports used to fill and file returns
    • HELB Contrubutions report
    • E-Payment register for Net salaries
    • Various user defined reports with bar graphs, tables, pie charts and line graphs
